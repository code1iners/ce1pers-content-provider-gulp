# Ce1pers Content Provider

Simple build automation project with Gulp.js.

## Main Tech Stacks

- Gulp.
- Each gulp plugins.
- TypeScript.
- Babel.

## How to play

- First at all, install dependencies.

  ```bash
  npm install
  ```

- Building statically contents.

  ```bash
  npm run build
  ```

- See distributed files in ./dist folder.
